# Web Para Autónomos – Conversión Directa

## Vision
Landing page española ultra-performante para autónomos que buscan más clientes y confianza. Transforma visitantes en leads vía WhatsApp, llamadas, formulario y Calendly. Target: trabajadores autónomos en España que necesitan presencia profesional y sistema de captación simple.

## Design
- `--primary: 207 87% 35%` (deep trust blue #0D5CA6)
- `--primary-dark: 207 77% 19%` (darker blue #083257)
- `--accent: 42 54% 51%` (gold reviews #C89B3C)
- `--accent-light: 42 87% 72%` (light gold #F4D27A)
- `--background: 0 0% 100%` (white)
- `--foreground: 207 25% 20%` (dark slate)
- `--muted: 0 0% 96%` (light grey)

Fonts: Plus Jakarta Sans (headings), Work Sans (body)
Style: Modern premium conversion-focused, mobile-first, very airy, strong hierarchy, direct not corporate

## Features
- Hero with dual CTAs (Calendly + WhatsApp), trust banner, limited offer card (497€ vs 697€)
- Before/After photo section showcasing professional transformation
- Social proof with Google 5.0 reviews (gold accent)
- Portfolio grid (6 real project placeholders)
- Pain points section addressing lead loss, slow response, trust issues
- Comparison table (Web Para Autónomos vs cheap web vs traditional agency)
- 3 pricing tiers (Esencial 497€, Pro 797€ highlighted, Premium 1297€)
- Final conversion CTA section
- Footer with scarcity messaging
